# composedb-playground

This is a simple playground for experimenting with Ceramic and and ComposeDB.

## Getting Started

Install dependencies:

```bash
pnpm install
```

Start the Ceramic daemon:

```bash
pnpm dev
```

Build the composite

```bash
pnpm build
```

Start the GraphQL server:

```bash
pnpm start:graphql
```
